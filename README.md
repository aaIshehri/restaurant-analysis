# 🍽️ Restaurant Menu & Order Analysis

## 📌 Project Overview

This project analyzes restaurant menu performance and customer ordering behavior using SQL.
The goal is to understand which items perform well, how customers order, and where the business can improve.

---

## 🎯 Objectives

### 1. Menu Analysis

* Explore menu items and categories
* Analyze pricing (min, max, average)
* Understand distribution across categories

### 2. Orders Analysis

* Analyze total number of orders and items
* Identify order patterns and sizes
* Detect large orders and customer behavior

### 3. Customer Behavior Analysis

* Combine menu and order data using JOIN
* Identify most and least ordered items
* Analyze top spending orders
* Explore patterns in high-value orders

---

## 🛠️ Tools Used

* SQL (Data Analysis)
* MySQL

---

## 📊 Key Insights

* Some items are highly popular while others have very low demand
* High-spending orders tend to include multiple items from specific categories
* Menu pricing varies significantly across categories

---

## 📁 Project Structure

* `analysis.sql` → Contains all SQL queries used in the analysis

---

## 💡 Conclusion

This project demonstrates how raw restaurant data can be transformed into actionable insights, helping improve menu design and customer experience.
